# UbiDesign Shower Component Development

These documents describe the procedures for building each component in the UbiDesign shower monitoring system

## Tools Needed: 

1. Solder
2. Liquid Flux
3. Soldering iron
4. hot air gun
5. Solder pot
6. Side cutters
7. Wire Strippers
8. Adjustable spanner
9. Pipe cutter
10. 25mm hole saw and arbor
11. Drill
12. Wireless things voyager serial interface
13. PC with drivers for voyager serial interface and arduino IDE installed
14. USB mini and miro cables
